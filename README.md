# SOT-Yggdrasil-Auth
SOT Yggdrasil Auth是一个轻量级、零依赖、单文件的Yggdrasil身份验证服务器方案，兼容 PHP 7.2+ 环境，开箱即用。它提供了完整的用户注册/登录、皮肤管理、后台管理等功能。

特性
🚀 单文件部署：仅需一个 index.php 文件，无需任何依赖
👤 用户系统：注册、登录、会话管理
🎨 皮肤系统：上传/删除皮肤，支持标准和纤细模型
🛡️ 安全机制：CSRF防护、请求速率限制、会话管理
📊 管理后台：内置现代化的 Web 管理界面，可管理用户、系统配置等
🔑 动态密钥：自动生成 RSA 密钥对用于签名
📦 零数据库：使用JSON存储用户数据、皮肤文件、系统配置等

PHP环境要求
PHP 7.2+

注意
使用前请将data目录设置为禁止外部访问
不建议在正式场景下使用此项目
